harmony-analyser

harmony-analyser is a tool for harmonic analysis, featuring chord transition tool for MIDI input and audio analysis from WAVE input. The application was developed 2013-2016 as a joint Master's thesis & PhD thesis project (Comenius University in Bratislava, Charles University in Prague).
